## 这一段测代码块

例如本地预览命令：

```bash
cd redcelery
hugo server --buildDrafts
```

浏览器访问终端里提示的地址（一般是 `http://localhost:1313/`）。

## 这一段测引用与链接

> 先把流水线跑通，再慢慢换主题、加分类和数学公式也不迟。

主题与配置仍在迭代中，若你之后改成纯中文站，只要在 `hugo.toml` 里把默认语言切到 `zh-cn` 即可。
